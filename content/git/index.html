How to use Git

This article outlines how to commit your code to git and push it to GitHub. The process is quite simple and you should get in the habit of doing it whenever you are done coding for the day.

Procedure

  1. Log on to JupyterHub Notebooks: https://jupyterhub.ischool.syr.edu with your SU NetID and password.
  2. Click the New button in the upper right and select Terminal from the menu to launch the terminal.
  3. Change into the library folder, type :
    cd ~/library
  4. Change into the git folder, type :
    cd 202 then press the [TAB] key to complete the folder name! Cool!
  5. Your prompt should now indicate you are in the repository folder. You can verify by typing:
    git status
    If you see fatal: not a git repository then you are not in the correct folder. Try the previous steps again.
  6. To make git track the changes you made, type :
    git add --all
  7. To make git save the changes you made, type :
    git commit -m "[[Your Message]]
    Of course, you don’t type [[Your Message]]!!! Instead you should enter a brief message about the work you’re saving. e.g. "Completed lab 2" or "still working on HW 3".
  8. To send your saved changes to Github, type :
    git push origin master

That’s it! You can view your work on the Github website if you like.
