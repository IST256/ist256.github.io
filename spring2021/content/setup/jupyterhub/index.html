Migrating to Jupyterhub

Before You Begin

  • Have the following information handy:
    • Your GitHub username and password.
    • Your GitHub Classroom repository URL (copy it from the github website).
    • The name of your Github repository. This comes after the IST256-classroom. part of your repository URL.
  • Login to azure notebooks and open the terminal:
    • commit any changes you have made:
      git add --all
      git commit -m "moving to jupyterhub"
    • push code to github:
      git push origin master

Procedure

  1. Visit: https://jupyterhub.ischool.syr.edu/.
  2. Click the Sign in with iSchool SSO button.
  3. Click the Login with your SU NetID button, then login.
  4. In about a minute your server will start. When it is finished you will see the Jupyter website.
  5. Open a terminal. New –> Terminal
  6. Clone your repository: git clone then paste in your close repository from the previous step. For example mine was https://github.com/IST256-classroom/2019-fall-learn-python-spring-mafudge. You will be prompted for your github username.
    You will be prompted for your github password. YOU WILL NOT SEE ANY CHARACTERS YOU TYPE.
    When you authenticate, the cloning process will start which takes about 1 minute.
  7. After you clone the repository move into the repository folder:
    cd FOLDER where folder is the name of your Github repository. You can start typing it and press the TAB key to complete it.
  8. From inside your repository folder, Type:
    git config credential.helper store to configure git to save passwords.
  9. Type: git config --global user.email "netid@syr.edu" to configure git to use your email. MAKE SURE TO USE YOUR SU EMAIL ADDRESS!
  10. Type: git config --global user.name "Your Name" to configure git commits with your name. AGAIN USE YOUR FULL NAME HERE e.g. Michael Fudge.

You are now setup on jupyterhub!!!
